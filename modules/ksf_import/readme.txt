WHAT DOES THIS MODULE DO?

This module ksf_import will import QIF and QSX/OSX files into Frontaccounting

Steps:
	
	

INSTALLATION:

1. FrontAccounting -> Setup -> Install/Activate Extensions

   Click on the icon in the right column corresponding to ksf_import

   Extensions drop down box -> Activated for (name of your business)

   Click on "active" box for ksf_import -> Update

2. FrontAccounting -> Setup -> Access Setup

   Select appropriate role click on ksf_import header and entry -> Save Role

   Logout and log back in

3. FrontAccounting -> TAB -> ksf_import

   Click on button -> XXX
 
   Fill in details
